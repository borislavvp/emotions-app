# Sandbox

## About

This repo contains a NextJS app for demoing and testing Hume APIs emotions detection. It implements several streaming interfaces that provide an intuitive validation of model quality.

## Requirements

- [Node](https://nodejs.org/)

## Development

```bash
$ npm install
$ npm run dev
```

Development mode will start serving on `localhost:3001`.

## Production

The sandbox deploys on Netlify on merge to the main branch.
